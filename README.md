# Sharie - Social Content Sharing Platform

A full-stack social media platform for sharing content, connecting with others, and real-time messaging. Built with a modern tech stack focusing on performance and user experience.


## Features

### Authentication & Security
- **JWT-based authentication** with access/refresh tokens
- Secure password hashing using bcrypt
- Protected routes and API endpoints
- Session management with token rotation

### Real-time Messaging
- Instant messaging with WebSocket connections
- Typing indicators and online status
- Message delivery and read receipts
- Chat history persistence

### Social Features
- User profiles with custom bios and avatars
- Content sharing (posts, images, links)
- Like, comment, and share functionality
- Follow/unfollow other users
- Activity feeds and notifications

### Modern UI/UX
- Responsive design for all devices
- Smooth animations and transitions
- Dark/light mode support
- Intuitive navigation and layout

## Tech Stack

### **Frontend**
- **React 18** with TypeScript
- **Vite** - Fast build tool and dev server
- **Zustand** - Lightweight state management
- **Tailwind CSS** - Utility-first styling
- **React Router** - Client-side routing
- **Axios** - HTTP client for API calls
- **Socket.io-client** - WebSocket client

### **Backend**
- **Express.js** - Node.js web framework
- **MongoDB** with **Mongoose** - Database & ODM
- **JSON Web Tokens (JWT)** - Authentication
- **Socket.io** - Real-time WebSocket communication
- **bcrypt** - Password hashing
- **multer** - File upload handling
- **cors** - Cross-origin resource sharing

### **Development & Deployment**
- **ESLint & Prettier** - Code quality
- **dotenv** - Environment variables
- **Postman** - API testing
- **Render/Vercel** - Deployment options
